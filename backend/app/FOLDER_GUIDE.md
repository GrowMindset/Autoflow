# App Folder

This is the main backend application package.

Files that belong here:
- FastAPI app setup
- business modules
- execution engine code
- routers, schemas, models, and services

Why this folder exists:
- To keep all runtime backend code in one place
- To make imports consistent across the project
- To separate application code from config, tests, and migrations

Subfolders inside this folder should each have a single clear responsibility.
